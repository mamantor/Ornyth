scrollFactorX :number

Description
The horizontal scroll factor of this Game Object. The scroll factor controls the influence of the movement of a Camera upon this Game Object. When a camera scrolls it will change the location at which this Game Object is rendered on-screen. It does not change the Game Objects actual position values. A value of 1 means it will move exactly in sync with a camera. A value of 0 means it will not move at all, even if the camera moves. Other values control the degree to which the camera movement is mapped to this Game Object.
