new DeathZone( source, killOnEnter )

Description
A Death Zone. A Death Zone is a special type of zone that will kill a Particle as soon as it either enters, or leaves, the zone. The zone consists of a `source` which could be a Geometric shape, such as a Rectangle or Ellipse, or your own object as long as it includes a `contains` method for which the Particles can be tested against.
Parameters
Name Type Description
source object An object instance that has a `contains` method that returns a boolean when given `x` and `y` arguments.
killOnEnter boolean Should the Particle be killed when it enters the zone? `true` or leaves it? `false`
