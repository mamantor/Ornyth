cameras/sprite3d/Camera.js, line 768

getPointSize( vec, size, out )

Description
This is a utility function for canvas 3D rendering, which determines the "point size" of a camera-facing sprite billboard given its 3D world position (origin at center of sprite) and its world width and height in x/y. We place into the output Vector2 the scaled width and height. If no `out` is specified, a new Vector2 will be created for convenience (this should be avoided in tight loops).
Parameters
Name Type Description
vec any The position of the 3D Sprite.
size any The x and y dimensions.
out any The result, scaled x and y dimensions.
