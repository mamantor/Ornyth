new BaseSoundManager( game )

Description
The sound manager is responsible for playing back audio via Web Audio API or HTML Audio tag as fallback. The audio file type and the encoding of those files are extremely important. Not all browsers can play all audio formats. There is a good guide to what's supported [here](https://developer.mozilla.org/en-US/Apps/Fundamentals/Audio_and_video_delivery/Cross-browser_audio_basics#Audio_Codec_Support).
