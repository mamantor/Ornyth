CanvasPool

Description
The CanvasPool is a global static object, that allows Phaser to recycle and pool Canvas DOM elements. This singleton is instantiated as soon as Phaser loads, before a Phaser.Game instance has even been created. Which means all instances of Phaser Games on the same page can share the one single pool
