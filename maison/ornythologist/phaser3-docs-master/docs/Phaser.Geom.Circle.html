new Circle( [ x [, y [, radius ] ] ] )

Description
A Circle object. This is a geometry object, containing numerical values and related methods to inspect and modify them. It is not a Game Object, in that you cannot add it to the display list, and it has no texture. To render a Circle you should look at the capabilities of the Graphics class.
