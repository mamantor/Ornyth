events/EventEmitter.js, line 114

animation( key [, url [, xhrSettings ] ] )

Description
Adds an Animation JSON file to the current load queue. Note: This method will only be available if the Animation JSON File type has been built into Phaser. The file is **not** loaded immediately after calling this method. Instead, the file is added to a queue within the Loader, which is processed automatically when the Loader starts.
Parameters
Name Type Attributes Description
key string | array | object A unique string to be used as the key to reference this file from the Cache. Must be unique within this file type.
url string <optional>
 URL of the file. If `undefined` or `null` the url will be set to `.json`, i.e. if `key` was "alien" then the URL will be "alien.json".
xhrSettings object <optional>
 File specific XHR settings to be used during the load. These settings are merged with the global Loader XHR settings.
Returns
The Loader.
Details
