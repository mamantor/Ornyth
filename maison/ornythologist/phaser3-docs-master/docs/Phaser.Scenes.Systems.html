new Systems( scene, config )

Description
The Scene Systems class. This class is available from within a Scene under the property `sys`. It is responsible for managing all of the plugins a Scene has running, including the display list, and handling the update step and renderer. It also contains references to global systems belonging to Game.
Parameters
Name Type Description
scene Phaser.Scene The Scene that owns this Systems instance.
